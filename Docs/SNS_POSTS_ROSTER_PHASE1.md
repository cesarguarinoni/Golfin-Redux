## 💡 Posting Tips

**Facebook:**
- Use longer-form, conversational tone
- Emphasize features & gameplay
- Mention timeline (April launch)
- Call-to-action: "Stay tuned!"

**Instagram:**
- Short, punchy captions
- Emojis for visual interest
- Hashtags: 5-8 relevant tags
- Each image tells one story
- Post carousel 3-5 images (one post, multiple slides)

**Cross-posting:**
- Same image can work on both FB & IG
- Adjust caption tone for platform
- Timing: Post when audience is active
