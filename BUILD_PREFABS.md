In Unity:
```
Tools → GOLFIN → Roster → Build Character Thumbnail Prefab
```

- `Assets/Prefabs/UI/Roster/CharacterThumbnailCard.prefab`
- Complete with:
  - Background (Image)
  - Portrait (Image)
  - Name Label (TextMeshProUGUI)
  - Rarity Badge (top-left)
  - Level Badge (top-right)
  - Selection Highlight (border)

### 2. Build StatBar Prefab

In Unity:
```
Tools → GOLFIN → Roster → Build StatBar Prefab
```

**What it creates:**
- `Assets/Prefabs/UI/Roster/StatBar.prefab`
- Complete with:
  - Icon (Image)
  - Label (TextMeshProUGUI)
  - Progress bar (fill)
